# Theme-Park-Stackathon
This a short term project to build a Full-Stack Application using the MERN Stack (FE)

## Collaborators: [Abeer Rozba](https://github.com/abeer-rozba), [Mohamed Hasan](https://github.com/mohammed-abadi), [Yasmeen Ebrahim](https://github.com/yasmeeneb1999)
***

### ***Description***
Theme Park Stackathon (lame) is an App built using React. It is a movie based theme park collection/library, With everybody having access to view add and delete theme parks.

***
### ***Getting Started***
You are instantly greeted with the Home Page. The home page consists of "Featured Parks" which will pick 3 random theme parks from the list that you can quickly access, there is a navigation bar for 3 pages,
Home, Theme Parks, and the Add Park form page. Theme Parks will list every current entry of the database. Add Park page will take you to the form page and you can fill it with the required fields in order to 
add a theme park to the list.

#### The project is deployed and can be accessed [here](http://themeparkstackathon.surge.sh/)
